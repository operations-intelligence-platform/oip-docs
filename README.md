# OIP Docs 
Documentation repository for the **Operations Intelligence Platform (OIP)**. 

## Purpose

This repository contains the technical documentation, architectural decisions, project vision, roadmap, data definitions, and planning material for OIP.

## Contents

- Project overview
- Architecture documentation
- Technical decisions
- Data model definitions
- KPI definitions
- Roadmap
- Meeting notes
- Research and references
